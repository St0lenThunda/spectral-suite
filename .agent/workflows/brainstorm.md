---
description: Brainstorm 3-5 examples with pros/cons and system cost analysis
---
1. Analyze the user's input to understand the core problem or requirement.
2. Brainstorm 3-5 distinct approaches or examples that solve the problem.
3. For each example, provide:
   - **Concept**: A brief description of the idea.
   - **Pros**: Advantages of this approach.
   - **Cons**: Disadvantages or risks.
   - **System Cost**: Estimated impact on performance, complexity, or maintenance (Low/Medium/High).
4. Present the analysis in a structured markdown format.
5. Ask the user for feedback or if they want to pursue one of the options.
